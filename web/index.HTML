mijn CheatSheet

  • cd..: previous folder
  • cd (foldername): naar een bepaalde folder gaan
  • git init: hiermee maak je een repository
  • git status: hieremee kan je zien wat de status is van de repository en of er al commits zijn gemaakt
  • git add (bestand naam)/ git add .: hiermee voeg je een bestand toe aan de staging erea. zodat je het later kan committen
  • git commit -m "(message)": hieremee laat je git het bestand volgen
  • git log: hieremee kan je zien wie de commit gedaan heeft, wanneer de commit gedaan is en de commit message
  • maak een repository op github -> je klikt op de groene knop (new) en je kopieert de commands en plakt ze in cmd
  • git remote add origin Dan koppel je de repository op je computer aan de repository in je github
  • git branch -m main: hij noemt de branch main (je kan main veranderen naar wat je wilt)
  • git push -u origin main: Dan koppel je de repository op je computer aan de repository in je github
  • git push: je stuurt alle commits naar github
